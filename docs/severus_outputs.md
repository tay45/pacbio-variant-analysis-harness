# Severus Outputs

Native Severus outputs are preserved. For the pinned 1.7 contract, discovery checks `all_SVs/severus_all.vcf`, conditionally `somatic_SVs/severus_somatic.vcf`, `breakpoint_clusters.tsv`, `breakpoint_clusters_list.tsv`, `plots/severus_*.html`, logs, optional auxiliary files, and unknown native files.

Output discovery writes JSON and Markdown inventories and treats missing or empty required outputs as failures.
