# Cohort Reporting

`cohort-report` writes `reports/cohort_report.md` from the cohort plan, status,
QC aggregation, storage estimate, and optional incremental comparison outputs.

The report includes the research-use disclaimer, cohort identifiers, analysis
modes, reference/tool summaries, Slurm plan, status counts, failure categories,
warnings, missing metrics, storage estimates, limitations, and operator actions.

Optional dependency-light HTML can be requested with `--html`.

Technical completion does not establish biological or clinical validity.
