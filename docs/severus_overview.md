# Severus Somatic SV Overview

Severus support is a research-use Phase 2F.1 planning, validation, and reporting layer for PacBio HiFi matched tumor-normal and explicitly authorized tumor-only structural variants. It is pinned to the official Severus 1.7 command/output contract and is separate from DeepSomatic SNV/indel logic, germline pbsv logic, and future CNV modules.

Standard tests use mocked commands and synthetic VCFs only. Full biological benchmarking on production tumor cohorts remains future work.
