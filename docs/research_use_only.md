# Research-Use-Only Statement

This workflow is for research use only. It is not a clinically validated,
diagnostic, or treatment-decision system.

Successful command execution, PASS QC status, or report generation does not
establish analytical validity, clinical validity, regulatory compliance, or
suitability for patient care.

QC thresholds are configurable research-use checks. They are not universal
acceptance criteria and must be validated for each project, sample type,
platform, tool version, and reference bundle.
