# Security Policy

This is research-use software. Do not submit patient data, credentials, private keys, tokens, institution-private data, or large sequencing files in issues, pull requests, or examples.

## Supported Version

The current public alpha line is `0.2.7a1`, prepared for tag `v0.2.7-alpha.1`.

## Reporting A Vulnerability

Use GitHub private vulnerability reporting if available. Otherwise, open a minimal public issue that avoids sensitive details and does not include exploit payloads, credentials, private data, or patient information.

No clinical safety claims are made for this software.
